Governance: Protección PII Inteligente

Clasificación Automática

  • 150+ clasificadores LLM integrados
  • Detecta NIF, IDs de empleados, números de póliza
  • Tags semánticos: PII_TYPE = 'TAX_ID'
  • Aplicación: 1 comando por columna

Masking Basado en Tags

  • Política única → aplica a TODOS los campos tagged
  • Role-based: ACCOUNTADMIN ve datos reales
  • Otros roles: ***MASKED***
